# My Health Coach
A health and fitness application made to easily track, plan and organize workouts and meals to aid in a much healthier lifestyle.

## Tech Stack

* Front-End:
  * Typescript
  * Next.js
  * TailwindCSS

* Back-End:
  * Express.js
  * Docker
  * CICD
  * PostgreSQL
